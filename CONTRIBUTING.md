# Contributing

## Branching
- main: protected, nur via PR mit grünem CI.
- feature/*: kleine, fokussierte Änderungen.

## Konventionen
- SwiftLint/SwiftFormat vor Commit.
- Tests zuerst/mitliefern, Coverage ≥ 90%.

## Tooling
- Optional: Tuist/XcodeGen, Fastlane.
- MCP (Codex CLI) konfiguriert unter `.codex/`.
