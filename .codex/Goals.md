# Goals
1) SwiftUI iPhone App "MathCrossword" mit adaptiver Schwierigkeit (Klasse 4).
2) Saubere Trennung: Puzzle-Engine (rein, testbar) vs. UI (SwiftUI).
3) Testabdeckung ≥ 90% (Unit + Snapshot/Integration).
4) CI-Pipeline (Build, Test, Coverage Gate, Lint).
5) Content: Aufgabenpools für 4. Klasse (add/sub/mul/div bis 1000, schriftliche Verfahren,
   Einheiten, einfache Bruchteile — didaktisch sinnvoll).
6) Lokalisierung: DE (Basis), später EN.
