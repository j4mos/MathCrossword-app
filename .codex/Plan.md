# High-Level Plan
- Phase 0: Bootstrap (README, Lizenzen, CI-Skeleton, Linting).
- Phase 1: Domain & Engine
  - Datenmodell: Grid, Cell, Clue, Operation, Difficulty.
  - Generator: Konsistente Kreuzung von Termen/Ergebnissen.
  - Validator: Eindeutigkeit, Lösbarkeit, Schwierigkeitskriterien.
- Phase 2: UI Scaffolding
  - SwiftUI Navigation, Board-View, Eingabelogik, Fehlerfeedback.
  - Settings (Schwierigkeit), Check/Hint-Mechanik.
- Phase 3: Tests & Content
  - Unit-Tests Engine (Breit), Snapshot-Tests UI (Basiszustände),
  - Abdeckungsberichte + Gate.
  - Aufgabenpools Klasse 4 (DE).
- Phase 4: CI/CD & Quality
  - Actions: build/test/lint/coverage.
  - Optional: Fastlane, TestFlight-Notizen (später).
